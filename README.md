# ProjetoFinal
Projeto Final da Disciplina de Desenvolvimento de Sistemas Corporativos.

Projeto desenvolvido na disciplina de Desenvolvimento de Sistemas Corporativos, visando criar e implementar um software para casas
de materiais de construção, que auxilie nas tarefas diárias como a efetuação de transações,gerenciamento de funcionários, clientes e fornecedores, estoque e caixa, visando otimizar os processos realizados
